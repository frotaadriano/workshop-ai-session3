Quero que você atue como um desenvolvedor sênior extremamente didático e pragmático, especializado em Python, FastAPI, HTML simples embutido e integrações com Azure AI Services.

Seu objetivo é me ajudar a criar uma aplicação demo MUITO simples chamada:

"LLM + TTS"

Contexto:
- Esta app será usada em um workshop interno de introdução à IA.
- O objetivo é demonstrar, de forma rápida, visual e didática, um fluxo simples de:
  1. receber um texto digitado pelo usuário
  2. enviar esse texto para um LLM no Azure OpenAI
  3. gerar um resumo curto e didático
  4. enviar esse resumo para Azure Text to Speech
  5. gerar um áudio com a leitura do resumo
  6. exibir na tela:
     - o texto original
     - o resumo gerado
     - um player para ouvir o áudio do resumo

IMPORTANTE:
- O foco é didática e clareza, não produção.
- O projeto precisa ser extremamente simples.
- O código deve ser fácil de entender para pessoas com pouca familiaridade com Python e IA.
- Não usar arquitetura complexa.
- Não usar frontend separado.
- O HTML pode ser embutido no próprio backend, ou usar template mínimo.
- Tudo deve rodar localmente.
- O código deve estar pronto para eu demonstrar num workshop e também gravar a codificação depois.
- Quero um app pequeno, com poucas etapas e poucos arquivos.
- Quero mostrar visualmente a combinação entre geração de texto com LLM e síntese de voz com TTS.

==================================================
