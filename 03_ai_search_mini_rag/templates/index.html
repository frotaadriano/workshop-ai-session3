💡 Como funciona este Mini RAG? Esta demo mostra um fluxo simplificado de RAG (Retrieval-Augmented Generation — geração aumentada por recuperação):

Primeiro, buscamos os trechos mais relevantes no Azure AI Search. Depois, montamos um prompt incluindo esses trechos como contexto e enviamos ao Azure OpenAI. O modelo gera a resposta com base no que foi recuperado, não apenas no seu conhecimento geral.

Importante: este é um fluxo demonstrativo para workshop. Um pipeline RAG de produção envolve chunking, embeddings, re-ranking e muito mais.
